# Portable Flux Workflows

The examples in this directory demonstrate core MatEnsemble workflow patterns that are portable across Frontier, Perlmutter, Pathfinder, Linux containers, and any other Flux-capable runtime.

Use these examples to understand how to structure `Pipeline` objects, chores, executable tasks, MPI chores, dependencies, and adaptive strategies. When running on a specific system, pair the Python workflow pattern here with that system's launch instructions and examples.
