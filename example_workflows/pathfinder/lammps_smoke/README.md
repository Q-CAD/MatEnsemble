# Pathfinder LAMMPS Smoke Campaign

This example runs a small CPU LAMMPS Lennard-Jones melt through MatEnsemble on Pathfinder.

Build or place a Pathfinder MatEnsemble Apptainer image in this directory as `matensemble.sif`, then run either:

```bash
bash run_interactive.sh
```

or submit:

```bash
sbatch run_batch.slurm
```
